整數格子點

社寵長在上課學到要算格子點的題目，但覺得一個一個數太慢了，所以來求助她妹妹たかめ少女xiplus。

輸入說明

第一行有一個數字$T$，接下來$T$行
每一行有6個數字$x_{1}~y_{1}~x_{2}~y_{2}~x_{3}~y_{3}$，為三個點的座標，皆為整數
$T \leq 10^{4} ； -10^{3} \leq x_{1},y_{1},x_{2},y_{2},x_{3},y_{3} \leq 10^{3}$

輸出說明

對於每行測資 輸出三角形內的格子點數(含邊界)。

範例

輸入輸出
1
3 7 -2 -2 7 -1		42

提示

寫的好的話，退化三角形也會AC

配分方法

$0\%$範例
$25\%$兩股平行$X,Y$軸的直角三角形；$-10 \leq x_{1},y_{1},x_{2},y_{2},x_{3},y_{3} \leq 10$
$25\%$兩股平行$X,Y$軸的直角三角形
$25\%$$-10 \leq x_{1},y_{1},x_{2},y_{2},x_{3},y_{3} \leq 10$
