E. 二進位數字

本題有測資，但正確性未知

婕兒正在做一台能將十進位數字轉成二進位數字的計算機，快來幫幫她吧。

給一個十進位數字，請轉成二進位數字，如果是二進位循環小數，請用括弧表示最短循環節，答案長度必須最短。

輸入說明

單筆測資 只有1個正數$N$，代表十進位數字，可能是整數或小數

輸出說明

輸出二進位

範例

輸入輸出

5

101

1.6

1.(1001)

提示

如果小數部分是 0110011001100...，答案 (0110) 會比 0(1100) 長度短。

配分方法

20%$0 \leq N \leq 1000$
20%$0 \leq N \leq 10^{19}$
20%$0 \leq N \leq 1000 ，N的十進位小數部分長度 \leq 1$
40%$0 \leq N \leq 10^{19} ，N的十進位小數部分長度 \leq 6$
