哭力怕的聖誕樹




遙遠的地方有一個哭力怕國，裡面住了許許多多的哭力怕：有大有小，有方有圓，據說還住著一隻T17屆的社寵（？）。哭力怕國裡，有成千上萬棵樹。

聖誕節快臨了，哭力怕國所有的樹都被裝飾成聖誕樹，聖誕節當天，一群來自喳喳國的外賓來參訪哭力怕國。為了招待訪客們，哭力怕國王決定親自帶他們參觀每一區的聖誕樹。

但是，天然呆呆的國王把圖鑑給搞丟了，只留下一張字條，記錄著每區的樹從樹頂到底依序的樣子。

每區的樹都長得不一樣，而且同一區的每棵樹長的高度也不太一樣。

請你依據這區的種類以及高度，告訴國王該聖誕樹的長相。
PS. 樹幹的部分，跟樹葉最中間底下一樣。

輸入說明
多筆測資，每筆測資有兩行。
第一行是一個字串S，代表樹頂到樹底依序的樣子。(1 <= S長度 <= 30)
第二行是一個整數N，代表樹的高度。(0 <= N <= 35)

輸出說明
對每一筆測資，輸出該聖誕樹的長相。

若N=0，請不要輸出任何東西。

若N > S長度，代表國王的字條太舊了，請輸出「Please Update!」(不含引號)。
輸出結果請參考範例輸出。

sample input
1KaLmIg2kB7f
3
5T
1
hZkykUu
8

sample output
  1
 1K1
1KaK1
  a
  a
  a
5
5
Please Update!

提示/注意事項
請不要在行尾輸出多餘的空白字元。
