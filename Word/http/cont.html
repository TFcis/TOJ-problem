$Word$

$Word$的列印功能只能選取起始與結束頁碼及張數，現在有一個檔案要列印，每頁需要印的數量不一樣，請規畫最佳的方法使得發送列印命令的次數最少。

另外因為紙有多一些，所以可以多印幾張是不需要的浪費來減少命令次數。

輸入說明

單筆測資 第一行有$2$個整數$P,N$，分別為頁數以及可浪費的張數 第二行有$P$的整數$A_1\sim A_P$，$A_i$代表第$i$頁要印幾張 $P \leq 1000 ； 0 \leq \forall A_i \leq 1000$

輸出說明

輸出最少須發送幾次命令

範例

輸入輸出說明

5 0 4 5 2 3 2

4

$P_1*4 + P_2*5 + P_{3\sim5}*2 + P_4*1$

5 1 4 5 2 3 2

3

$P_{1\sim2}*5 + P_{3\sim5}*2 + P_4*1$ 此時$P_1$多印一張

5 3 4 5 2 3 2

2

$P_{1\sim2}*5 + P_{3\sim5}*3$ 此時$P_{1,3,5}$各多印一張

提示

配分方法

$99\%$$A,B < 10$
$1\%$$A,B < 100$
