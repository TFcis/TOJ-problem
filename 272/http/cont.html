整理桌面

由於社寵長還是把桌面檔案隨意擺放，搞得亂亂的，終於有一天她妹妹たかめ少女xiplus生氣了，決定將桌面的東西全部刪掉，由於檔案實在太多了，一次刪除會導致檔案總管當掉，所以必須分批刪除。
多個檔案同時刪除時，必須是能夠一次框選起來的檔案，所以請幫幫她找出刪除的方法數。

輸入說明

單筆測資。
第1行有3個數字$K,W,H$，分別代表有多少圖示、畫面寬度、畫面高度。
接下來有$K$行，每一行有2個整數$X,Y$，代表圖示座標。
$K < 10000 ； 0 < W,H \leq 10000 ； 0 < X \leq W ； 0 < Y \leq H$

輸出說明

輸出刪除的方法數。

範例輸入 1

4
2 2
1 1
1 2
2 1
2 2

範例輸出 1

61

範例輸入 2

4
3 2
1 1
1 2
2 1
3 2

範例輸出 2

63
