C. 文件索引

本題目前只有範測

亦晨拿著小穹依著ASCII順序整理好的文件要送去服務台，沒想到笨拙的亦晨跌了一跤，把文件的順序都弄亂了，但小穹用她神奇的瞳術看了就發現亂掉的文件卻形成另一種新的編號順序，但為了要懲罰亦晨，便要她自己找出順序，「快去給我找出來，信不信我揍妳」小穹心裡暗自地想。

輸入說明

多筆測資 每筆第一行有1個整數$N$，接下來$N$個字串 字串由數字和大小寫英文字組成 $N = 0$表示輸入結束 $0 \leq N \leq 30000$

輸出說明

對於每筆測資，輸出新的字母順序(保證有唯一解)

範例

輸入輸出說明

5 DCD BDDD BBBBC BBBBA C 0

DBCA

從所有字串第1個字元得知D<B<C 從第3,4個字串第5個字元得知C<A

提示

亦晨是小說「前進吧!! 高捷少女日常」新出現的人物，為捷運站的新人，小穹帶領的後輩。

配分方法

25%$N \leq 10$
25%$N \leq 1000$
50%題目範圍
