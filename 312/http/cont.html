Codeforces 625D. Extra

Put simply, how many answers to Codeforces 625D?

$Xiplus$ competed in Codeforces Round #342 (Div. 2) and gained +105 rating points taking place 121. It's a impressive success. He had solved out the problem D, so he asked a new problem, that is much harder.

Let's denote a flip operation of an integer as follows: refer OEIS A004086 and Codeforces 625D.

$Xiplus$ picked some number $a$ without leading zeroes, and flipped it to get number $a_{r}$. Then he summed $a$ and $a_{r}$, and told his sister $Jill$ the resulting value $n$. Her goal is to find $ALL$ valid $a$.

Because $Jill$ didn't solved the original problem, she didn't know how to do. Come and help her.

Input

The first line of the input contains a single integer $n~(1 \leq n \leq 10^{9})$.

Output

Print the number of $a$ that $a + a_{r} = n$.
