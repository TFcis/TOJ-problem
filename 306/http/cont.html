有緣千里來相會，無緣對面不相逢

本題有測資，但正確性未知

到了一年好幾度的出門比賽(社遊)時間了，其中的故事就不細數了。時間來到在台北車站要準備回台南，在一陣混亂之中，小穹和婕兒走散了XD(?)，於是她們借助艾咪的新店(?)感應Lv.5知道兩人在哪裡，於是決定要在一個她們能夠同時到達的地方，快來幫幫她們吧。PS.她們前往集合地點都會走最短路徑。

給定一張連通的簡單圖及兩點$A,B$，求出距離$A$最短距離等於距離$B$最短距離的所有點。

輸入說明

單筆測資 第一行有兩個正整數$N,S$，代表有$N$個點、$S$條邊 第二行有兩個正整數，代表$A,B$的點編號 接下來有$S$行，每行有$3$個正整數$C,D,L$，代表$C$和$D$之間有一條長度為$L$的邊 $N \leq 10^{5} ； S \leq min( \frac{N(N-1)}{2} , 2 \times 10^{5} )； 1 \leq A,B,\forall{C},\forall{D} \leq N ； 0 < \forall{L} \leq 10^{7}$

輸出說明

輸出所有符合的點 格式請輸出$E~F~P$ 若$E = F ； P = 0$，代表點在原本的頂點$E$上 若$E < F$，代表點在$\overline{EF}$上，距離$E$的$P\%$（$P$四捨五入到整數位） 若邊上所有點都是答案，輸出這條邊的兩端點 輸出請先依$E$再依$F$由小到大排序
